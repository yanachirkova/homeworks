# Инструкция для запуска docker контейнера с простым R-скриптом

1. Создайте на своем сервере, где установлен docker, отдельную папку для работы с контейнером.

2. Скачайте и скопируйте туда все файлы из этой папки на github-репозитории.

3. В рабочей папке создайте еще одну папку под названием `results`. Туда будут сохранены результаты работы скрипта.

4. Откройте терминал и перейдети в рабочую директорию.

5. Запустите следующую команду:
   ```bash
   docker build -t myimage .
   ```
       
   Эта команда собирает образ для doker-контейнера из файла Dockerfile и присваивает ему тэг myimage. Важной деталью является местонахождение Dockerfile. Если он будет сохранен не в текущей директории, то выполнение команды приведет к ошибке.

6. Затем запустите следующую команду:
   ```bash
   docker run -ti --rm -v "$PWD"/results:/home/results -w /home --user root myimage
   ```
   
   Такая команда создает контейнер по образу myimage и запускает его в интерактивном режиме с последующим удалением. Она привязывает папку results директории контейнера к созданной вами такой же папке на вашем сервере. Следовательно, все созданные и сохраненные в процессе запуска контейнера файлы будут также сохранены в этой папке.
